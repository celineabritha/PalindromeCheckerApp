# Palindrome Checker App

## Objective
The Palindrome Checker App is a console-based Java application developed to validate whether a given string is a palindrome using multiple programming approaches. The project strengthens core Java fundamentals, data structures, algorithms, and design patterns.

---

## Implemented Use Cases

### UC1 – Application Entry & Welcome Message
Displays application information when the program starts.

### UC2 – Hardcoded Palindrome Check
Checks palindrome result using a predefined string.

### UC3 – String Reverse Method
Reverses a string using loops and compares with the original.

### UC4 – Character Array Palindrome Check
Uses a character array and two-pointer technique.

### UC5 – Stack-Based Palindrome Checker
Validates palindrome using Stack (LIFO principle).

### UC6 – Queue + Stack Comparison
Demonstrates FIFO vs LIFO behavior.

### UC7 – Deque-Based Palindrome Checker
Compares characters from front and rear efficiently.

### UC8 – Linked List Palindrome Checker
Uses LinkedList operations for validation.

### UC9 – Recursive Palindrome Checker
Implements palindrome checking using recursion.

### UC10 – Case-Insensitive & Space-Ignored Check
Ignores spaces and letter case during validation.

### UC11 – Internal Stack / Array Method
Uses internal array structure for palindrome logic.

### UC12 – Strategy Pattern Implementation
Dynamically selects palindrome algorithms using polymorphism.

### UC13 – Performance Comparison
Compares execution time of different palindrome algorithms using `System.nanoTime()`.
